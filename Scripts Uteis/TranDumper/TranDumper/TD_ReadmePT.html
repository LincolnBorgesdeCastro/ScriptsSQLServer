Oque é TranDumper?
TranDumper é um FREEWARE desenvolvido para ajudar você a ler o braindumps que você encontrar.
Este programa converte os braindumps (e outros .txt's) para um formato que permita a você utilizá-lo como se fosse um programa de simulados. Qualquer modificação feita no TranDumper (código ou dump text file) deve ser liberado como freeware.

Você pode ajudar a traduzir a interface de usuário do TranDumper. Os arquivos a serem traduzidos (TD_Readme.html, Trandumper.html) devem ser colocados em uma pasta com 2 letras representando o país.
No momeno, Trandumper esta pronto para : DE, FR, SP, IT.

Obrigado por usar TranDumper.


Como formatar o arquivo texto para o TranDumper?
Para exemplo de como usar o TranDumper, você pode ver um diretório de exemplo. Você pode ler braindumps com pouca ou nenhuma modificação para o dumps. Veja o sample_plain.txt para uma explicação do arquivo de dump, ou veja sample_enhanced.txt que mostra o uso de características mais avançadas.

Você somente deve realizar alguma alterações na formatação do dump para ter certeza de que o TranDumper lerá ele apropriadamente. Você simplemente necessita salva a página web do dump como plain text (*.txt) e então verificar a formatação do seu arquivo dump como abaixo:
  1. O assunto deve começar com um número e um ponto em uma nova linha.
    EXEMPLO: 1.  or  99. (Número máximo de questões: 400)

  2. Uma linha em branco é obrigatória entre o texto da questão e respostas com múltipla-escolha.

  3. As opções de resposta devem começar com caracter alfanumérico e um ponto ou a ")".
    EXEMPLO: a.  a)  A. (Número máximo de respostas: 15)

  4. RESPOSTAS DEVEM ESTAR EM UMA LINHA.
    O formato das respostas deve ser escrito como se segue para ser corretamente reconhecido.
    EXEMPLO: Ans:a,c   Answer: b    Ans : bc   Answer: b,c and d

  5. Se existir qualquer explicação ou comentário para a resposta, uma linha em branco é  obrigatória entre a linha da última opção de resposta e a explicação ou comentário
  6. Todas as opções de seleção e respostas são case-insensitive. Isto é, ambos Ans: e ANS: são sintaxes aceitas.


Como executar o TranDumper?
Requisitos: Internet Explorer 5.0 ou superior e resolução de vídeo 800x600. TranDumper somente pode ser executado em browsers compatíveis com o IE. Ele usa um ActiveX (File System object of scripting engine para abrir o arquivo local). E também porque o Trandumper.html contém muitos VBscripts.
  1. Garanta que você possui Internet Explorer 5.0 ou superior instalado. TranDumper NÂO PODE rodar no IE 3.x OU 4.x.

  2. Instale o TD na pasta local, cada dump deve estar em uma subpasta. Por favor de uma olhada no exemplo que ve mcom o TD.

  3. Porque ele deve abrir arquivos locais, é recomendado que a configuração de segurança do seu browser seja configurada para médio ou baixa. Pode ocorrer do Internet Explorer pedir a confirmação para confiar no ActiveX que estiver rodando. Para prosseguir você deve cliar em "Yes".

  4. Não use o PWS (Personal Web Server) ou IIS para rodar o TD (TranDumper). O diretório de instalação não pode ser qualquer diretório  "Virtual" do IIS ou PWS.

  5. O TD somente pode ser executado localmente. TD não roda com um web site.
